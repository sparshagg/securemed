# Workflow: First Session Bootstrap

Use this when starting the repository from scratch.

## Steps

- [ ] Read `AGENTS.md`, `README.md`, `ROADMAP.md`, `PLAN.md`, `SECURITY.md`, `DATABASE.md`.
- [ ] Inspect repository files.
- [ ] Confirm whether Python package skeleton exists.
- [ ] If no skeleton exists, create minimal package only.
- [ ] Add `pyproject.toml` with required dependencies.
- [ ] Add fake sample EMR JSON.
- [ ] Add the first crypto utility and tests.
- [ ] Run tests.
- [ ] Update README commands to match reality.
- [ ] Commit and push.

## First Code Target

The first working slice should be AES-GCM encrypt/decrypt, ECDSA sign/verify, pytest tests, and no API yet unless crypto utilities are complete.
